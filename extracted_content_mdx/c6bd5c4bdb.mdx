# FelisÂ¶

Felis provides a system for describing astronomical data catalogs with a user-friendly YAML data format. It covers not only the metadata required to create database schemas, but also a wide range of additional metadata associated with IVOA standards and other applications. While developed for the Vera C. Rubin Observatory, which utilizes the framework for describing its Science Data Model Schemas, Felis is designed to be general enough to be used by any astronomical data provider.
