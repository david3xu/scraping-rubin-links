# __ DocuShare Help

## Sort Results

The order in which objects are displayed in the search results:

  * **Ranked Order** displays objects that contain all of the search text before objects that contain only one or two of the words.
  * **By Title** displays objects alphabetically by title.
  * **By Date** displays objects chronologically, with most recently modified objects displayed first.
  * **By Date Ascending** displays objects chronologically, with most recently modified objects displayed last.
  * **By Type and Title** groups objects by type and orders objects alphabetically by title.
  * **By Type and Date** groups objects by type and orders objects chronologically, with most recently modified objects displayed first.
  * **By Title Reversed** displays objects in reverse alphabetical order by title.
  * **By Owner** displays objects alphabetically by owner name.
