# ctrl_bps_panda

`ctrl_bps_panda` is a package in the LSST Science Pipelines.

It provides a PanDA plugin for LSST PipelineTask execution framework, based on `ctrl_bps`.

  * SPIE paper from 2022: The Vera C. Rubin Observatory Data Butler and Pipeline Execution System.
  * User Guide.
