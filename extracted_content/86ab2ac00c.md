### UsageÂ¶

The primary classes are:

  * `MTDomeCsc`: controller for the Simonyi Survey Telescope dome.

  * `MockDomeController`: simulator for the dome TCP/IP interface.




Run the `MTDome` controller using `bin/run_mtdome.py`.
