rsp.lsst.io
===========

[](#rsplsstio)

This is the source repository for the Rubin Science Platform's documentation website, [https://rsp.lsst.io](https://rsp.lsst.io). The RSP's documentation is a [Sphinx](https://www.sphinx-doc.org/en/master/) project. The configuration is provided by Rubin's [Documenteer](https://documenteer.lsst.io/guides/index.html) package.

Contributing
------------

[](#contributing)

You can improve this documentation by sending a pull request or creating an issue. See the [Contributing](https://rsp.lsst.io/contributing/index.html) section of the docs for details.
